# Contributing to FLWR Branding Studio Kit

Thank you for your interest in contributing! This project aims to democratize high-level brand strategy using AI agents.

## How to Contribute

1.  **Fork the repository**
2.  **Create a feature branch** (`git checkout -b feature/amazing-feature`)
3.  **Commit your changes** (`git commit -m 'Add amazing feature'`)
4.  **Push to the branch** (`git push origin feature/amazing-feature`)
5.  **Open a Pull Request**

## Guidelines

*   **Templates:** If adding new templates, ensure they are Markdown-formatted and placed in `.agent/templates/`.
*   **Rules:** Updates to `branding_agent_rules.md` should follow the "Context Shielding" and "Reality Check" protocols.
*   **Scripts:** Python scripts should be platform-independent.

## Reporting Issues

Please use the GitHub Issues tab to report bugs or suggest enhancements.
