## Como Usar no Figma

1. **Copiar o conteúdo** de cada arquivo `.md`
2. **Colar no plugin de importação** do Figma (quando desenvolvido)
3. O plugin deve criar automaticamente:
   - Text layers com os títulos (H1, H2, H3)
   - Text layers com os parágrafos
   - Listas formatadas (bullets)
   - Tabelas (para o Dicionário, se necessário)

## Formato Padrão

Todos os arquivos seguem a convenção:
- **Títulos H1 (#)**: Seção principal (ex: "Posicionamento")
- **Títulos H2 (##)**: Subseções (ex: "Propósito Central")
- **Títulos H3 (###)**: Detalhes (ex: "Tom")
- **Parágrafos**: Texto corrido sem formatação especial
- **Listas**: Bullets com `-` ou numeração com `01.`
